发表于 2017-03-19 | 分类于 博客搭建

前言

历经千辛万苦，我又用Hexo+GitHub搭建了一个新的博客。为了方便自己以后犯错误的时候能快速重新搭建（估计不会），也为了给一些后来者做一个参考，所以我打算写下这篇文字教程。

准备

因为这个博客是用Hexo+GitHub搭建的，所以我们需要先下载Git和Node.js，并且进行安装。然后在GitHub上注册一个账号。还需要有一个文本编辑器，我用的是sublime text 3。sublime text是一个比较轻便的文本编辑器，它也有许多的插件支持，当然你也可以依据自己的喜好使用其他编辑器。

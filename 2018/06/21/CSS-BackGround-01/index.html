CSS背景（一）

发表于 2018-06-21 | 分类于 WEB前端

最近看了一阵子书，看了些前端的东西，特别是《CSS揭秘》这本书，看了之后受益颇多。虽然还只看了前几章，但我已经被书中对CSS的运用技巧所惊叹。所以打算在这里写一些自己在书中学到的和自己琢磨到的一点东西。

描边背景

就是在div的周围画出类似与订装线一样的虚线，提高视觉感受。主要用到以下几个CSS属性：

border
outline
outline-offset

主要思路是先利用border属性视觉上增大背景大小，然后用outline属性画出虚线，并且将outline-offset属性设置为负值，将虚线向内移动达成订装线的视觉效果。效果如下图：

主要代码：

background: skyblue;
border: 10px skyblue solid;
border-radius: 10px;
outline: 1px dashed white;
outline-offset: -10px;

条纹背景

条纹背景很好理解，就是由不同颜色的条纹填充的背景，主要用到了background: linear-gradient()和background: repeating-linear-gradient。

横向和竖向条纹

横向条纹和竖向条纹是比较好实现的两种条纹背景，横向条纹依靠linear-gradient(color1 , color2)的渐变特性，只要将color1的终止位置和color2的起始位置设置为同一个值，颜色就会在一个无限小的区域渐变转换，从效果上来看就是颜色直接发生了转变，而竖向条纹则只要在linear-gradient()中设置一个90°的旋转就好了，效果如下图：

主要代码：

.d1{
    background: linear-gradient(skyblue 50%, pink 50%);
    background-size: 100% 50px;
}
.d2{
    background: linear-gradient(90deg, skyblue 50%, pink 50%);
    background-size: 50px 100%;
}

斜向条纹1

斜向条纹其实也是由横向条纹旋转而来，但是仅仅像设置竖向条纹那样简单的设置一个角度的话就会出现下面图片中第一个背景图一样的效果，会发现跟想象中的不太一样，他们不能很好的拼接在一起，要实现第二个背景的效果需要进行更多的设置来将他们拼接在一起。

主要代码：

.d1{   //与想象中的效果不太一样
    background: linear-gradient(45deg, skyblue 50%, pink 50%);
    background-size: 50px 50px;
}
.d2{
    background: linear-gradient(45deg, skyblue 0, skyblue 25%,
        pink 0, pink 50%,
        skyblue 0, skyblue 75%,
        pink 0, pink 100%);
    background-size: 100px 100px;
}

斜向条纹2

为了实现斜向条纹，用上面那种方式比较复杂，可以用另一个属性repeating-linear-gradient()来简化代码而实现相同的效果。
主要代码：

background: repeating-linear-gradient(45deg,
    skyblue, skyblue 35px,
    pink 0, pink 70px);

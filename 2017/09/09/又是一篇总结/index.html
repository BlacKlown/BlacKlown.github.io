半年更的自我总结

发表于 2017-09-09 | 分类于 自我总结

转眼又是半年过去了，上个学期又犯了和以前一模一样的错误：开学时给自己订的计划又没能实施。来到大学到现在似乎只学会了怎么花爸妈的钱，自己其实一直都想快快经济独立，至少能给父母减轻负担，但其实感觉我到现在还是没有真正的学会去为自己做打算。

开学一周了，还没学到些东西，就又在心里给自己列下了一堆购物清单，想换电脑了，想买平板，想换键盘，想买台NS，总之总是觉得别人的东西比自己的好，而自己又什么都缺。但其实我在这方面也还算懂事，很多东西会先跟父母商量再做决定，可能最后不会立项拨款，但是心里总是还是会有念想，只能等到以后经济独立以后再自己买来作为礼物送给自己吧。

现在笔记本用的是联想拯救者，为什么想换。因为续航真的是游戏本的通病，还有明显的发热和高出其他笔记本很多的重量。两年前的我追求配置买下了这台电脑，但人是会变的，我已经不是那么爱玩游戏了，并且这台电脑原配的无线网卡是没有Linux版本的驱动的，我更希望能够拥有一台能够长时间续航并且更加便携的电脑，现在有很多轻薄本都装有940或者150显卡，它们就是现在的我的目标。但是换一台笔记本需要的也不是一笔小钱，我也跟妈妈简单的交谈过这个问题，得到的是意料之中的否定。换电脑应该是大学毕业经济独立之后才有可能的事情了，现在也就只能在心里憧憬一下了。

其实从上个学期开始我就一直想要一台Ipad mini了，其实科技产品、数码产品对理工男应该有莫名的吸引力。如果在条件允许的情况下，每个理工男都会想用数码产品给自己全副武装上吧。不过这个估计也只能在我经济独立之后才能买了吧。

原来用的键盘，或者应该说现在还是在用的键盘是IKBC-C87黑青。不过上学期末不知道怎么就进水了，烧坏了空格键，前两天寄去北京维修了。来回邮费都要50多，大概是我两天多的伙食费了，还不知道修要多少钱。所以寄去之前就考虑不如直接换一把新的，就看上了POKER，同时也觉得60%的键盘应该能够制约下我爱玩游戏的性子吧。然后又发现GANSS前两天出新产品了，ALT61，跟POKER一样是60%的键盘，199的预售价，顿时眼睛都直了。跟社团里说了下自己的想法，大家肯定都是清一色的回“想什么，买买买”。但其实我自己知道这不是必需品，键盘这种东西同一时间就能用到一把，买多了其他的就要吃灰，或者出掉，更何况是花父母的钱，一时间就很迷茫，不知道该怎么办。

再就是NS，任天堂出的新一代游戏机，在这方面，任天堂就没搞砸过。都要溢出屏幕的游戏性全变成你说服自己去买它的理由，不过还好，一从家里来到学校我似乎就没有了购买它的想法，不过可能哪天这个想法又会从心底冒出来吧。

大三了，就需要面临一个很大的问题：考研还是不考研。家里人都是希望我去考研的，我也知道考研的重要性，但是我心里是不想考研的。大学两年下来，我也算是大概弄懂了自己是个什么样的人吧，我知道自己的天赋绝对不低，但是最大的问题就是我是一个厌学的人，或者说是讨厌学校教育的人。我每天宁愿发呆也不愿去学习，等大学毕业，我更希望去找一份工作为父母分担一些辛苦。父母生下我的时候就不年轻了，现在也已经都50多岁了，我大学毕业再读完研究生还需要5年时间，我不想让他们再等下去。也有可能是我现在没认清自己的水平，才觉得能够毕业找到工作去分担父母的辛苦吧。

这个学期的每门课我基本上都上了一遍了，对这些课也有一些了解了，心中也有一些偏重了。我真的不愿去听那些老师讲得很枯燥的课，有一个老师甚至连自己的ppt都不能照着读通顺。

下个学期不会在学校待多久，学校就会把我们派去实习。相当与我只有差不多这一个学期能够好好学习了，不然就只能一张白纸走上社会了。所以还是要给自己定下目标和学习计划吧，虽然可能又会半途而废，但是总比没有的好。想学的东西有点多也不知道该怎么开始，所以就按兴趣来吧：

1.Python Linux
2.JS C#
3.php java

其实能够学到两门我都觉得很不错了，现在能够做到弄懂一点的也就是JS和java吧，但是Python对我总是有一种莫名的吸引力，再是对Linux也是有非常高的兴趣，正好这个学期也开了这门课，就好好学吧。JS应该是我自学得最多的语言了吧，并不是很想丢下它，这个学期还开了C#，老师也是非常有趣，抓住这个机会学习一下吧。

这个学期开学以来还是有好事发生的，起码我比以前勤快一些了，作息也规律许多了，但是有的老师还是能把我催眠掉，哈哈！
